# Security Policy

Copyright (c) 2026 Lee Mercey. Owner: Cortex Evolved Systems.

Please report security issues privately to the repository owner.

Do not publish exploit details in public issues. Include:

- affected commit or release
- operating system
- steps to reproduce
- expected impact
- any local files touched during reproduction

This repository is a public-review/demo package. It must not contain live user
runtime data, datasets, count files, model secrets, API keys, or restricted
payloads.
